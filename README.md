# 🏨 AtliQ Hospitality Analysis Dashboard

## 📌 Project Overview

This project analyzes the performance of AtliQ Hotels using Power BI. It provides insights into revenue trends, occupancy, pricing strategy, and hotel performance across different cities and time periods.

---

## 🎯 Business Objectives

* Track weekly revenue trends
* Analyze RevPAR (Revenue per Available Room)
* Compare weekday vs weekend performance
* Identify top-performing hotel properties
* Understand booking platform impact

---

## 🛠 Tools & Technologies

* Power BI
* Excel
* Data Visualization & Analysis

---

## 📊 Key KPIs

* **Revenue:** 377M
* **RevPAR:** 6.34K
* **Occupancy Rate:** 57.98%
* **ADR:** 10.93K
* **DSRN:** 653
